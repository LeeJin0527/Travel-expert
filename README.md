# Travel Expert
#여행지 추천어플

1. 국내와 해외를 선택하세요.
2. 원하는 항공권 가격대를 선택해주세요.
3. 유명한 관광지 또는 호캉스로 여행컨셉을 고르세요.
4. 최적화된 장소를 찾아드립니다.
5. 혼자 가기 쓸쓸하시다면, 동반자를 구하세요! 
6. 여행지에서의 파티를 구하시나요?
7. 근처에 가까운 파티 장소를 찾아드릴게요. 
8. Travel Expert에서 지상최고의 경험을 해보세요 !
